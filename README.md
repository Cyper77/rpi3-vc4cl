# VC4CL for Raspberry Pi 3.

## How to Install

### VC4C

```
./021_install_vc4c_runtime.sh
./022_install_vc4c.sh
```

### VC4CL

```
./021_install_vc4c_runtime.sh
./022_install_vc4c.sh
```

## How to Build

### pre-requirements

```
./000_clone_repos.sh
./001_gcc-6.sh
./010_install_khronos_icd_loader.sh
./011_install_opencl_header1.2.sh
./012_install_clang.sh
```

### VC4C

```
./020_build_vc4c.sh
```

### VC4CL

```
./030_build_vc4cl.sh
```
